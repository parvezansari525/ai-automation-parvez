# 🚀 AI Automation Project – SWIFT Message Classifier

## 📌 Overview
This project demonstrates an **AI-powered automation system** for classifying banking SWIFT messages into categories like Payment, Transfer, Refund, and Salary.

It simulates real-world **Intelligent Automation in Banking (BFSI)** using Python.

---

## 🎯 Features
- ✅ SWIFT message processing using Pandas
- ✅ AI-like classification using NLP-based scoring logic
- ✅ JSON output generation
- ✅ Modular Python project structure
- ✅ Ready for LLM (AI) integration

---

## 🧠 AI Logic
The system uses a **hybrid AI approach**:
- Keyword-based NLP
- Scoring-based classification
- Designed to be extended with real AI (LLM APIs)
