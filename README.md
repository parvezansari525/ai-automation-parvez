# AI Automation - SWIFT Parser

## Overview
This project processes SWIFT-like financial messages and classifies them into categories using Python.

## Features
- Reads CSV data
- Cleans and processes messages
- Classifies transactions (Payment, Transfer, Salary, Refund)
- Outputs structured JSON

## Tech Stack
- Python
- Pandas

## Future Scope
- Replace rule-based logic with AI (LLM)
- Build real-time API
- Deploy on cloud
